# A collection of some [Tampermonkey](https://www.tampermonkey.net/) scripts

## clockify-expected-hours

A script that calculates the expected hours on the [Clockify reports page](https://app.clockify.me/reports/).
It's not very sophisticated: it multiplies 8 hours with the number of days you booked hours for.

## zep-import

As neither the usage of [ZEP](https://www.zep.de/) REST API nor its Excel import is possible as of now, this script is a
workaround: it copies entries from a JSON file to ZEP via the Web UI. The JSON file is generated by
the [JsonV1 export](../tw-core/src/main/kotlin/dev/hertlein/timesheetwizard/core/exporting/domain/service/strategy/JsonV1.kt).
